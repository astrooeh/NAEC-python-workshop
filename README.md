# NAEC-python-workshop
Tutorial zum NAEC Austria Python Workshop 
*Autoren: Desmond Grossmann, Nicolas Muntean, Paul Beck*

*Dieses Tutorial stellt einige einfache Möglichkeiten mithilfe von Python und offenen astronomischen Datensätzen einfache astrophysikalische Problemstellungen für den Unterricht und Abschlussarbeiten zu erarbeiten. Es kann, unter Angabe der Quelle, jederzeit frei verwendet werden. Stand der Informationen: 20. Juni 2026*
